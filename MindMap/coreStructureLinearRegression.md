# Core for linear regression

## Formula
- f(x) = $W^TX+b$
- error: $\varepsilon$ = y-f(x)

## loss function
- MSE(Mean Squared Error)

## Optimizer
- Gradient Descent
  - Stochastic Gradient Descent(随机梯度下降, SGD)
  - Standard Gradient Descent

## Regularization and overfitting
  - L1/L2 Regularization

## Evaluation
  - R²/RMSE
